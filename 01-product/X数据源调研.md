# X(Twitter)热点数据源调研报告

## 1. 官方X API

### 1.1 API定价方案 (2025-2026)

| 套餐 | 价格 | 读取请求 | 发布请求 | 趋势访问 |
|-----|------|---------|---------|---------|
| **Free** | $0 | 不可用 | 1,500/月 | ❌ |
| **Basic** | $100/月 | 10,000 | 50,000 | ❌ |
| **Pro** | $5,000/月 | 1,000,000 | 300,000 | ✓ |
| **Enterprise** | $42,000/月起 | 定制 | 定制 | ✓ |
| **Pay-Per-Use** | 按量付费 | 按实际使用 | 按实际使用 | ✓ |

### 1.2 新增按量付费模式 (2025年底推出)

| 操作 | 单价 |
|-----|------|
| 读取推文 | 按次计费 |
| 查看趋势 | 包含在基础费用中 |
| 关注关系查询 | $0.01/次 或 $10/1000次 |

**优点**: 灵活，适合低频用户
**缺点**: 仍需审核批准，接入门槛较高

## 3. MCP服务 (推荐用于Agent)

### 3.1 可用的Twitter MCP Server

| 服务 | 功能 | 安装方式 |
|-----|------|---------|
| **Twikit Twitter Tool** | 发推、搜索、情感分析 | `smithery install` |
| **Twitter MCP Server** | 发推、搜索、回复、OAuth | 标准MCP安装 |

### 3.2 MCP服务优势

```
┌─────────────────────────────────────────────────────────────┐
│                    MCP vs 传统API对比                         │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  传统API                          MCP服务                    │
│  ───────                          ──────                    │

│  • 复杂的OAuth流程                • 即插即用                 │

│  • 高昂费用                      • 社区驱动，免费或低成本   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## 5. 推荐方案

### 5.1 MVP阶段 - MCP服务优先

**推荐**: 使用Twitter MCP Server

**原因**:
1. 零成本启动
2. 快速集成
3. 适合Agent架构
4. 社区活跃

**风险**:
- 社区维护，稳定性未知
- 可能有速率限制
- 数据准确性待验证

### 5.2 成长阶段 - 混合方案

```
┌─────────────────────────────────────────────────────────────┐
│                      混合架构设计                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  主数据源           备份数源           降级方案               │
│  ────────           ────────           ────────               │
│  MCP服务            Apify            缓存数据                 │
│     │                  │                  │                  │
│     ├─ 优先使用         ├─ MCP失败时切换   ├─ 无网络时         │
│     ├─ 零成本           ├─ 付费但可靠      ├─ 历史热点         │
│     └─ 快速响应         └─ $10-50/月       └─ 延迟<5分钟       │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 5.3 成熟阶段 - 根据量级选择

| 月活用户 | 推荐方案 | 预估成本 |
|---------|---------|---------|
| < 100 | MCP + 缓存 | $0 |
| 100-1,000 | MCP + Apify备份 | $10-50/月 |
| 1,000-10,000 | Apify为主 | $50-200/月 |
| > 10,000 | X API Pro或自建 | $5,000+/月 |
