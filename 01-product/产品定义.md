# XTrendAI 产品定义文档 v2.0

## 1. 产品定位

### 1.1 产品名称
XTrendAI - X(Twitter)热点驱动的任务生成Agent

### 1.2 产品愿景
成为连接社交媒体热点与用户场景的智能桥梁，将X上的实时资讯转化为可执行的并行任务列表。

### 1.3 核心价值主张
- **个性化热点** - 基于地区、年龄、种族的个性化热点推荐
- **场景化任务** - 将热点转化为特定场景的可执行任务
- **即插即用** - 生成的任务可直接复制使用（如提示词、脚本等）

### 1.4 目标用户
| 用户类型 | 描述 | 核心需求 |
|---------|------|---------|
| POD商家 | 按需打印电商卖家 | 根据热点生成设计提示词 |
| 内容创作者 | 博主/YouTuber | 根据热点找内容创作方向 |
| 营销人员 | 社媒运营/广告投放 | 根据热点做营销素材 |
| 开发者 | 需求快速响应的开发 | 根据热点做小工具/爬虫 |

## 3. 个性化热点策略

### 3.1 地区差异化

| 地区 | 热点特征 | 数据来源 |
|-----|---------|---------|
| 美国 | 娱乐、体育、政治、科技 | X Trends US |
| 中国大陆 | 科技、娱乐、国际新闻 | X Trends + 中文标签 |
| 欧洲 | 政治、体育、文化 | X Trends by Country |
| 全球 | 节假日、大型赛事、国际新闻 | X Trends Worldwide |

### 3.2 年龄差异化

| 年龄段 | 热点偏好 | 算法权重 |
|-------|---------|---------|
| 18-24 | 娱乐、游戏、网红 | Pop Culture + Gaming |
| 25-34 | 科技、职场、生活方式 | Tech + Career |
| 35-44 | 财经、教育、健康 | Finance + Education |
| 45+ | 健康、时事、传统媒体 | News + Health |

### 3.3 公共热点（全覆盖）

无论用户画像如何，以下热点始终展示：
- **节假日** - 全球主要节日
- **大型赛事** - 奥运会、世界杯、超级碗
- **重大新闻** - 国际突发事件
- **游行活动** - Pride Week、黑人生存运动等
