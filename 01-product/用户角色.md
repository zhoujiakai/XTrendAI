## 2. 角色定义

### 2.1 访客 (Guest)

#### 基本信息
| 属性 | 说明 |
|-----|------|
| 角色名称 | 访客 |
| 用户类型 | 未注册用户 |
| 主要诉求 | 快速体验产品 |

#### 权限矩阵

| 功能 | 权限 | 限制 |
|-----|------|------|
| 访问页面 | ✓ | - |
| 查看热点 | ✓ | 仅限全球公共热点(前3个) |
| 查看任务 | ✓ | 仅限POD场景 |
| 复制任务 | ✓ | 每日3次 |
| 用户画像配置 | ✗ | 需注册 |
| 场景选择 | ✗ | 固定POD场景 |
| 历史记录 | ✗ | - |

#### 用户故事
> 作为一个路过的网友，我想快速看看现在有什么热点，不需要注册就能体验一下。

---

### 2.2 免费用户 (Free User)

#### 基本信息
| 属性 | 说明 |
|-----|------|
| 角色名称 | 免费用户 |
| 用户类型 | 注册用户 |
| 主要诉求 | 个性化体验 |

#### 用户画像配置

| 配置项 | 可选值 | 说明 |
|-------|-------|------|
| 界面语言 | 🇨🇳中文 / 🇺🇸English | 单选，默认自动检测 |
| 地区 | 🇺🇸美国 / 🇨🇳中国 / 🇬🇧英国 / 🇯🇵日本 / 🌏全球 | 单选 |
| 年龄段 | 18-24 / 25-34 / 35-44 / 45+ | 单选 |
| 种族 | [可选] 亚裔 / 非裔 / 拉丁裔 / 白人 | 单选 |
| 场景 | POD / 内容创作 / 营销 / 开发 | 多选 |

#### 权限矩阵

| 功能 | 权限 | 限制 |
|-----|------|------|
| 访问页面 | ✓ | - |
| 查看热点 | ✓ | 个性化热点(前10个) |
| 查看任务 | ✓ | 所有选择的场景 |
| 复制任务 | ✓ | 每日20次 |
| 用户画像配置 | ✓ | - |
| 场景选择 | ✓ | 最多2个场景 |
| 历史记录 | ✓ | 最近7天 |
| 热点刷新 | ✓ | 每小时1次 |
| API访问 | ✗ | - |

#### 用户故事
> 作为一名POD卖家，我想根据自己的地区和年龄获取更精准的热点，每天生成一些设计灵感。

### 2.3 Pro用户 (Pro User)

#### 基本信息
| 属性 | 说明 |
|-----|------|
| 角色名称 | 专业用户 |
| 用户类型 | 订阅付费用户 |
| 主要诉求 | 高频使用、全场景覆盖 |

#### 用户画像配置

| 配置项 | 可选值 | 说明 |
|-------|-------|------|

| 年龄段 | 多选 | 可覆盖多年龄段 |

#### 权限矩阵

| 功能 | 权限 | 限制 |
|-----|------|------|
| 访问页面 | ✓ | - |
| 查看热点 | ✓ | 全部热点，无限刷新 |
| 查看任务 | ✓ | 所有场景 |
| 复制任务 | ✓ | 无限制 |
| 用户画像配置 | ✓ | 多地区/年龄段 |
| 场景选择 | ✓ | 全部场景 |
| 历史记录 | ✓ | 永久历史 |
| 热点刷新 | ✓ | 实时刷新 |
| 自定义模板 | ✓ | 编辑任务模板 |
| 批量导出 | ✓ | 一键导出所有任务 |
| API访问 | ✓ | API密钥 |
| 优先支持 | ✓ | 专属客服 |

#### 用户故事
> 作为一名全职内容创作者，我需要实时获取热点并快速生成多个场景的任务，通过API集成到我的工作流。

## 5. 用户生命周期

```
        访问网站
           │
           ▼
     ┌──────────┐
     │  Guest   │
     │  免费体验 │
     └─────┬────┘
           │
        感兴趣？
      ┌────┴────┐
      │ 是      │ 否
      ▼         ▭
  ┌─────────┐   离开
  │  Free   │
  │  注册使用│
  └────┬────┘
       │
    频繁使用？
  ┌────┴────┐
  │ 是      │ 否
  ▼         ▭
┌─────────┐  保持Free
│  Pro    │
│ 付费升级│
└─────────┘
```

### 转化策略

| 触发时机 | 转化策略 |
|---------|---------|
| 复制次数达80% | "今日配额即将用完，升级Pro无限使用" |
| 连续使用7天 | "您已是忠实用户，Pro首月5折" |
| 尝试访问Pro功能 | "此功能需要Pro，立即升级" |
| 场景需求>2个 | "升级Pro解锁全部场景" |

## 6. 用户细分

### 6.1 按地区细分

| 地区 | 用户特征 | 热点偏好 |
|-----|---------|---------|
| 美国 | 英语为主，POD成熟 | 娱乐、体育、政治 |
| 中国 | 中英双语，电商活跃 | 科技、娱乐、国际 |
| 欧洲 | 多语言，文化多元 | 政治、体育、艺术 |
| 其他 | 英语为主 | 本地+全球热点 |

### 6.2 按年龄细分

| 年龄段 | 用户特征 | 热点偏好 |
|-------|---------|---------|
| 18-24 | 学生/初入职场，爱尝鲜 | 娱乐、游戏、网红 |
| 25-34 | 职场中坚，有消费力 | 科技、职场、生活方式 |
| 35-44 | 事业稳定，家庭为主 | 财经、教育、健康 |
| 45+ | 经验丰富，关注时政 | 健康、新闻、传统媒体 |

### 6.3 按场景细分

| 场景 | 典型用户 | 核心诉求 |
|-----|---------|---------|
| POD | 电商卖家 | 设计提示词 |
| 内容创作 | 博主/YouTuber | 创作方向、脚本 |
| 营销 | 运营/广告投放 | 文案、标题 |
| 开发 | 开发者 | 需求描述、技术方向 |
