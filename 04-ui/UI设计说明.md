# XTrendAI UI设计说明 v1.0

## 1. 设计概述

### 1.1 设计理念
- **简洁至上** - 去除冗余元素，聚焦核心内容
- **信息层次** - 通过视觉层次引导用户注意力
- **即时反馈** - 每个操作都有清晰的视觉反馈
- **国际化友好** - 支持中英文无缝切换

### 1.2 目标设备
- **桌面端**：1920px / 1440px / 1280px
- **平板端**：768px - 1024px
- **移动端**：375px - 768px


## 3. 组件设计

### 3.1 按钮 (Button)

#### 主按钮
```
背景: #0066FF
文字: #FFFFFF
高度: 40px
内边距: 0 24px
圆角: 4px
悬停: 背景变深 #0052CC
```

#### 次按钮
```
背景: 透明
边框: 1px solid #E5E7EB
文字: #1F2937
悬停: 背景 #F9FAFB
```

#### 文字按钮
```
背景: 透明
文字: #0066FF
悬停: 文字变深 #0052CC
```

### 3.2 输入框 (Input)

```
高度: 40px
内边距: 0 12px
边框: 1px solid #E5E7EB
圆角: 4px
聚焦: 边框变蓝 #0066FF + 外阴影
```

### 3.3 选择器 (Select/Checkbox/Radio)

#### 复选框
```
未选中: 白色背景 + 灰色边框
选中: 品牌蓝背景 + 白色对勾
悬停: 边框变蓝
```

#### 单选框
```
未选中: 白色背景 + 灰色边框
选中: 品牌蓝填充 + 白色圆点
```

#### 下拉选择
```
与输入框样式一致
下拉面板: 白色背景 + 阴影 + 圆角
```

### 3.4 卡片 (Card)

```
背景: #FFFFFF
边框: 1px solid #E5E7EB
圆角: 8px
内边距: 16px
阴影: Shadow-sm
悬停: Shadow-md
```

### 3.5 标签 (Tag/Chip)

```
高度: 24px
内边距: 0 8px
圆角: 12px
字体: 12px
```

### 3.6 提示 (Toast)

```
位置: 页面顶部居中
背景: #1F2937
文字: #FFFFFF
圆角: 8px
内边距: 12px 16px
阴影: Shadow-lg
动画: 从上滑入
```

## 6. 交互设计

### 6.1 微交互

| 交互 | 效果 |
|-----|------|
| 按钮悬停 | 背景色变化 + 轻微上移 (1px) |
| 按钮点击 | 缩放 (0.98) + 涟漪效果 |
| 卡片悬停 | 阴影加深 + 轻微上移 |
| 复制成功 | Toast提示 "已复制到剪贴板" |
| 加载中 | 旋转动画 + 骨架屏 |
| 切换开关 | 滑动动画 + 颜色渐变 |

### 6.2 页面过渡

```
页面切换: 淡入淡出 (300ms)
热点展开: 高度动画 (200ms)
侧边栏: 滑入滑出 (250ms)
模态框: 缩放 + 淡入 (200ms)
```

## 10. 可访问性

### 10.1 对比度
- 正文文字: 最小 4.5:1
- 大文字: 最小 3:1
- 交互元素: 最小 3:1

### 10.2 键盘导航
- Tab: 焦点移动
- Enter/Space: 激活按钮
- Esc: 关闭模态框/下拉

### 10.3 屏幕阅读器
- 所有图标有 aria-label
- 表单元素有关联的 label
- 状态变化有 aria-live
