## 4. 阻塞问题清单

### 🔴 BUG001 - API服务端localStorage访问错误 (严重)

| 属性 | 内容 |
|-----|------|
| 问题描述 | `/api/trends` GET请求返回INTERNAL_ERROR |
| 根本原因 | 服务端无法访问浏览器的localStorage |
| 影响范围 | 首页热点列表无法加载 |
| 负责人 | 全栈开发工程师 |
| 优先级 | **P0 - 阻塞** |

**修复方案**:
- 方案A：API不依赖localStorage，改为从请求头获取用户信息
- 方案B：使用cookies替代localStorage
- 方案C：Guest用户直接返回公共热点，不检查配额

### 🟡 BUG002 - 任务API找不到热点 (重要)

| 属性 | 内容 |
|-----|------|
| 问题描述 | `/api/tasks` 返回TREND_NOT_FOUND |
| 根本原因 | 依赖BUG001修复 |
| 负责人 | 全栈开发工程师 |
| 优先级 | **P1** |

### 🟡 BUG003 - UserService服务端调用localStorage (重要)

| 属性 | 内容 |
|-----|------|
| 问题描述 | UserService在服务端调用localStorage |
| 根本原因 | 同BUG001 |
