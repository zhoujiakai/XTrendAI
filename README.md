# XTrendAI - X(Twitter)热点驱动的任务生成Agent

## 项目概述

XTrendAI 是一个连接社交媒体热点与用户场景的智能桥梁，通过抓取X(Twitter)上的实时资讯，将其转化为可执行的并行任务列表。

## 核心价值

- **个性化热点** - 基于地区、年龄、种族的个性化热点推荐
- **场景化任务** - 将热点转化为特定场景的可执行任务（POD设计、内容创作、营销文案、功能开发）
- **即插即用** - 生成的任务可直接复制使用（如提示词、脚本等）
