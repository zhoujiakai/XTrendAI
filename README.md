# XTrendAI - Twitter热点驱动的设计生成平台

## 项目概述

XTrendAI 是一个自动化设计与按需打印(POD)平台，通过抓取Twitter热点话题，利用AI文生图模型自动创作设计作品，并将其应用到服装、水杯、窗帘等实体商品上。

## 核心价值

- **热点驱动** - 实时捕捉Twitter热门趋势，快速响应市场需求
- **AI创作** - 自动化设计生成，降低设计成本
- **按需打印** - 零库存模式，降低商业风险
- **多品类支持** - 支持服装、家居用品等多种商品类型
