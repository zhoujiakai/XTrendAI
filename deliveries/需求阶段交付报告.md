## 2. 产品概述

### 2.1 产品定位

**XTrendAI** 是一个 **X(Twitter)热点驱动的任务生成Agent**，通过抓取X上最新的资讯，生成适合于特定场景的并行任务列表。

### 2.2 核心价值

| 价值维度 | 描述 |
|---------|------|
| **个性化热点** | 基于地区、年龄、种族的个性化热点推荐 |
| **场景化任务** | 将热点转化为特定场景的可执行任务 |
| **即插即用** | 生成的任务可直接复制使用（如提示词、脚本等） |

### 2.3 核心流程

```
X热点采集 → 个性化筛选 → 场景映射 → 并行任务生成 → 用户复制使用
   │           │           │            │              │
 X API/    地区/年龄/   POD/内容/   热点1→[任务1,2,3]  nano banana等
 MCP服务    种族筛选     营销/开发    热点2→[任务1,2]    平台使用
```

## 5. 支持场景 (MVP)

### 5.1 POD场景 - 按需打印设计

**输出任务类型**: 图像生成提示词

**示例**:
```
A trendy t-shirt design for Super Bowl 2026,
featuring a stylized football helmet with team colors,
minimalist vector art style, suitable for print-on-demand
```

**使用方式**: 复制 → 粘贴到nano banana等AI图像生成平台

### 5.2 内容创作场景

**输出任务类型**: 创作方向/大纲

**示例**: YouTube视频选题、脚本大纲、话题标签

### 5.3 营销场景

**输出任务类型**: 广告文案/标题

**示例**: 促销标题、广告描述、营销标签

### 5.4 开发场景

**输出任务类型**: 功能需求描述

**示例**: 小程序需求、爬虫功能、查询工具

## 6. 个性化热点策略

### 6.1 地区差异化

| 地区 | 热点特征 |
|-----|---------|
| 美国 | 娱乐、体育、政治、科技 |
| 中国大陆 | 科技、娱乐、国际新闻 |
| 欧洲 | 政治、体育、文化 |
| 全球 | 节假日、大型赛事、国际新闻 |

### 6.2 年龄差异化

| 年龄段 | 热点偏好 |
|-------|---------|
| 18-24 | 娱乐、游戏、网红 |
| 25-34 | 科技、职场、生活方式 |
| 35-44 | 财经、教育、健康 |
| 45+ | 健康、时事、传统媒体 |

### 6.3 公共热点（全覆盖）

无论用户画像如何，以下热点始终展示：
- **节假日** - 全球主要节日
- **大型赛事** - 奥运会、世界杯、超级碗
- **重大新闻** - 国际突发事件
- **游行活动** - Pride Week等

## 7. 业务流程

### 7.1 用户配置流程

```
访客 → 注册/登录 → 画像配置向导 → 选择地区 → 选择年龄段 → 选择种族(可选) → 选择场景 → 保存配置
```

### 7.2 热点获取流程

```
用户点击「获取最新热点」 → 系统请求X API/MCP → 获取原始热点 → 个性化筛选 → 场景任务映射 → 返回热点+任务列表
```


## 8. 竞争分析要点

### 8.1 竞品对比

| 竞品 | 类型 | 对比 |
|-----|------|------|
| Google Trends | 热点追踪 | 无X数据、无任务生成 |
| TweetDeck | Twitter管理 | 无个性化筛选、无任务生成 |
| PromptBase | 提示词库 | 静态资源、无热点驱动 |

### 8.2 差异化定位

> **我们是唯一一个"X热点驱动 + 个性化筛选 + 多场景任务生成"的工具**
